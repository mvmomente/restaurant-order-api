# Casos de Uso

## Cliente

### UC001 - Visualizar Cardápio

O cliente pode consultar os produtos disponíveis.

### UC002 - Criar Pedido

O cliente pode criar um novo pedido.

### UC003 - Personalizar Pedido

O cliente pode adicionar ou remover itens de um produto.

### UC004 - Consultar Status do Pedido

O cliente pode acompanhar o andamento do pedido.

---

## Atendente

### UC005 - Criar Pedido

O atendente pode registrar pedidos para clientes.

### UC006 - Visualizar Pedidos

O atendente pode consultar os pedidos existentes.

### UC007 - Atualizar Status do Pedido

O atendente pode alterar o status do pedido.

---

## Cozinheiro

### UC008 - Visualizar Pedidos em Preparo

O cozinheiro pode consultar pedidos pendentes de produção.

### UC009 - Atualizar Status de Preparo

O cozinheiro pode informar que o pedido está sendo preparado ou concluído.

---

## Gerente

### UC010 - Cadastrar Produto

O gerente pode cadastrar novos produtos.

### UC011 - Editar Produto

O gerente pode alterar informações dos produtos.

### UC012 - Remover Produto

O gerente pode remover produtos do cardápio.

### UC013 - Visualizar Pedidos

O gerente pode acompanhar todos os pedidos.

---

## Administrador

### UC014 - Cadastrar Usuário

O administrador pode criar usuários.

### UC015 - Editar Usuário

O administrador pode alterar informações dos usuários.

### UC016 - Remover Usuário

O administrador pode remover usuários.

### UC017 - Gerenciar Perfis de Acesso

O administrador pode definir permissões para cada perfil.
